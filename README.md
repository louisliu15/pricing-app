# Overview

A demo project for calculating the total cost of an order with unit test cases.

## Dependencies

* node v12.16.2
* npm 6.14.4

# Local Deployment

## Install dependencies

`npm install`

## Build Application
Under project directory, run the following command in command line:

`npm run build`

## Run Application
Under project directory, run the following command in command line:

`npm run start {item-number} {price} {province}`

Example: `npm run start 3600 2.25 MI`, expected output is: "$7984.98"

## Test Application
Under project directory, run the following command in command line:

`npm run test`
